🌫️ Fog Dustry

«"When the fog consumed the world — the factories kept running."»

📖 About the Mod

Fog Dustry is a large-scale expansion mod for Mindustry V8 Build 153+ featuring:

- 🌫️ Fog biomes
- ⚙️ Advanced machinery
- 🔋 New power systems
- ☣️ Dangerous substances
- 🤖 Unique units
- 🏭 Atmospheric factories
- 🌌 New planets and sectors
- 🔬 Futuristic technologies

The world has fallen into a dense technological fog.
Old factories are rusting away.
AI systems have gone out of control.
And energy is now extracted from unstable fog matter...

---

✨ Features

🌫️ Fog Technology

A completely new fog mechanic:

- Fog can hide enemy units
- Some turrets can detect targets through the mist
- Special generators process fog energy
- Fog affects machine efficiency

---

⚡ Advanced Power Systems

Added:

- Fog Reactors
- Cryo Batteries
- Plasma Nodes
- unstable energy networks

Power is no longer just wires.
One mistake — and half the base turns into a crater 💀

---

🤖 Units

The mod introduces new unit classes:

Type| Description
Recon| Fog reconnaissance units
Fog Crawlers| Mutated ground machines
Plasma Flyers| Air assault units
Heavy Siege| Slow heavy destroyers
